---

Jedes größere Betriebssystem verfügt über eine Art Barrierefreiheit-API, die
es Bildschirmlesern ermöglicht, auf wichtige Informationen über die Objekte
und Ereignisse zuzugreifen, die dem Benutzer zur Verfügung stehen. Alle APIs
sind unterschiedlich, haben aber alle das gleiche Ziel: das Betriebssystem -
und die darauf laufenden Anwendungen - für Menschen mit Behinderungen zugänglich
zu machen. Eine API für Barrierefreiheit kann auch als Aufhänger für Programmierer
dienen, die Aufgaben automatisieren oder andere Operationen durchführen wollen,
die nicht mit Barrierefreiheit zu tun haben.
