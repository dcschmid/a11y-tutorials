## Seite suchen

Manchmal kann es schwierig sein, Informationen auf einer Webseite zu finden, wenn
man einen Screenreader benutzt. Wenn es schwierig ist, Informationen zu finden,
können Screenreader-Nutzer Suchbefehle verwenden, um bestimmte Informationen oder
Wörter auf einer Webseite zu finden. In der folgenden Tabelle sind die Suchbefehle
aufgeführt, die für bestimmte Screenreader verwendet werden können.

<table>
  <caption>Suche innerhalb einer Webseite</caption>

    </tr>
  </thead>
