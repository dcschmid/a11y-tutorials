## Inklusives Design

Eine Designmethode, die darauf abzielt, die gesamte Bandbreite menschlicher Vielfalt
zu ermöglichen und zu berücksichtigen, einschließlich eines breiten Spektrums an
Fähigkeiten und Behinderungen. Das Hauptziel ist es, einen einheitlichen Designansatz
zu schaffen, der es ermöglicht, mit verschiedenen Methoden auf dieselben Funktionen
zuzugreifen.

## Barrierefreiheit im Internet

Die Eigenschaften, die ein Web-Erlebnis für eine möglichst große Gruppe von
Nutzern zugänglich machen. Barrierefreiheit ist das Endergebnis eines
integrativen Designprozesses.

## Benutzerfreundlichkeit

Die Qualitäten, die ein Web-Erlebnis intuitiv und einfach zu nutzen machen. Ein
benutzerfreundliches Webdesign entspricht dem Zweck, für den die Website erstellt
wurde.

Um die Definition von Barrierefreiheit im Internet zu präzisieren, werden in
den <a href="https://www.w3.org/TR/WCAG21/" target="_blank">Content
Accessibility Guidelines (WCAG) 2.1</a> vier Hauptprinzipien der
Barrierefreiheit im Internet aufgeführt:

- Wahrnehmbar - Die Informationen und Komponenten der Benutzeroberfläche müssen
  für die Nutzerinnen und Nutzer auf eine Weise darstellbar sein, die sie
  wahrnehmen können.

- Bedienbar - Die Komponenten der Benutzeroberfläche und die Navigation müssen
  bedienbar sein.

- Verständlich - Informationen und die Bedienung der Benutzeroberfläche müssen
  verständlich sein.

- Robust - Die Inhalte müssen so robust sein, dass sie von einer Vielzahl von
  User Agents, einschließlich assistiver Technologien, zuverlässig interpretiert
  werden können.
