Ein guter Ausgangspunkt für manuelle Tests ist die Bewertung der Eingabemethoden.
In diesem Abschnitt werden alle Testmethoden vorgestellt, die bei der Prüfung der
Barrierefreiheit von Tastaturen eingesetzt werden. Auch wenn der Schwerpunkt des
Abschnitts Eine grundlegende Testroutine auf der Testphase des Entwicklungszyklus
liegt, sollten Entwickler/innen die Barrierefreiheit von Tastaturen bereits in
der Erstellungsphase testen, wenn sie ihren Code schreiben und Webinhalte erstellen,
um so viele geräteabhängige Fälle wie möglich zu erfassen, bevor ihr Code in der
Testphase bewertet wird.
